## Installation (Windows)

1. Clone or download this repo.
2. Run **`setup.exe`**, this auto-detects your system architecture and installs the layout.
3. Go to **Settings → Time & Language → Language & region**.
4. Under **English (United States)**, click **⋯ → Language options**.
5. Under **Keyboards**, click **Add a keyboard** and select **Rain**.
6. Switch to it anytime using **Win + Spacebar**.

> If "Rain" doesn't appear right away in the Win+Spacebar switcher, sign out and back in (or restart) to refresh the list.

## Uninstalling

1. Go to **Settings → Time & Language → Language & region**.
2. Under **English (United States) → Language options → Keyboards**, remove **Rain**.
3. Optionally, uninstall it from **Settings → Apps** as well.
